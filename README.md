# All Clear

A light minecraft mod for sweeping item entity regularly on server (in test now)

Used on server-side, but also available on local game, and most of the parameters support customization

### The mod can do:

*regularly cleanup item entity in server

*custom interval time between each cleanup via config file

*custom hint file via config file

*trigger a cleanup via command /allclear now

*detect the server's TPS, and trigger a mandatory cleanup when the TPS too low, the threshold value can be customed via config file

### The mod is still in test now and only support on Minecraft-1.21.1-neoforge version
